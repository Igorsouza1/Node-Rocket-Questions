# RocketQuestions

## Projeto
Uma aplicação de intereação com perguntas e resposta onde é possível criar uma sala para serem feitas perguntas anônimas, com o dono da sala podendo exluir e marcar perguntas como ldias.

## Tecnologias
- HTML
- CSS
- JavaScript
- EJS
- Express
- SQlite
